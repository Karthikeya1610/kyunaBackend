# Kyuna Jewellery Backend Setup Guide

## Prerequisites

- Node.js (v16 or higher)
- MongoDB
- npm or yarn

## Installation Steps

1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` file with required environment variables
4. Start the server: `npm run dev`
